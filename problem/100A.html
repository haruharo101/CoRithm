100A - 정수 덧셈

시간 제한 : 1초

메모리 제한 : 512MB

문제 설명

정수 $a, b$ 가 주어진다. 두 정수의 합을 출력한다.

입력

첫 번째 줄에 정수 $a, b$ 가 공백으로 구분되어 주어진다. $(1 \lt a, b \lt 10^{4})$

출력

두 정수의 합을 한 줄에 출력한다.

입출력 예제

입출력 1

입력 출력
1 1 2

입출력 2

입력 출력
1000 1234 2234

Co-Rithm

Co-Rithm은 Coding의 Co와 algoRithm의 Rithm을 합친
